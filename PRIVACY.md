## Data Collection

This extension collects **no personal data** and **no browsing data**.

## How It Works

The extension reads product prices and unit sizes that are already visible on the page at `www.ah.nl`. It computes the price per kilogram or liter locally in your browser and displays the result on the page. That's it.

## Network Requests

This extension makes **zero network requests**. All processing happens entirely within your browser.

## Permissions

The extension only requires content script access to `www.ah.nl`. It does not request any additional browser permissions.

## Third Parties

No data is shared with any third party.
